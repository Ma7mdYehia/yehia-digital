# yehia.digital

Personal profile website, digital CV, executive portfolio, and case-study library for Mohamed Yehia.

## Current milestone

**First screen only.** This build covers:

- Visual system (palette, typography, glass panels, motion)
- Left floating icon navigation (desktop) with mobile hamburger
- Hero section with placeholder headshot frame
- Proof / stats strip with four key metrics
- Quiet placeholder blocks for all remaining sections

## Tech stack

- **Next.js 15** App Router
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (subtle entrance animation only)
- No backend. No database.

## Content policy

All copy is sourced from approved documents only. No invented claims, no fabricated metrics, no stock people photography. HÄLSA rule respected: never "sourdough bread" — only "natural sourdough fermentation" where relevant.

## Run locally

```bash
cd 02_PORTFOLIO_WEBSITE/site
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
